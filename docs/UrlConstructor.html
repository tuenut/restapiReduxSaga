Class: UrlConstructor

UrlConstructor(resourcePath)

Class provides some methods to construct URI-string for REST API actions. Some resources can have extra actions like this: For api/v1/task/1/complete :

  • api/v1/ - is base URL,
  • task - is resource,
  • 1 - is entity ID,
  • complete - is extra action to complete task.
